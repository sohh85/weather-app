# お天気チェッカー

https://sohh85.github.io/weather-app/


# 概要
前後一週間分の天気を表示するサービスです。
東京・大阪・名古屋の天気を下記APIから取得しています。
https://www.metaweather.com/api/


# 修正点
- 過去の天気一週間分を表示
- 選択肢をボタンに変更
- bootstrap-vueで装飾
